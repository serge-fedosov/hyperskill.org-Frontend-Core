Description

In this stage, we are going to work on the about section. This section will be similar to the home section; it will contain an image, a paragraph, and a header.

Let's get a cool font from Google Fonts and apply it to our texts and headers to make your website even better. Use the !important CSS rule for the font family to override other styles. Add the Google font to HTML with <link> and use it in your CSS file.

Create a new file for the CSS styles called style.css and link the CSS file with the HTML file.

The CSS rule for the font family should be inside the style.css.

In this stage, we are working on the about component ID.

The layout of the HTML body for this stage should follow the structure below:

<body>
<header.../>
<main>
    <div (container)>
        <div id=home.../>
        <div id=about (container)>
            <div (container)>
                <img.../>
            </div>
            <div (container)>
                <h2.../>
                <p.../>
            </div>
        </div>
    </div>
</main>
<script (Bootstrap)/>
</body>
Objectives

In this stage, your website should contain:

  1. A <h2> tag;
  2. A <p> tag with a dummy text;
  3. A cafe-related image;
  4. A font from Google Fonts.
  5. A new style.css file, linked to HTML.
