Description

In this stage, we will develop the home section. It will be the first thing clients see on your website. This section will feature a big header and a cool image to capture clients' attention. Our home section would be similar to the Jumbotron component from Bootstrap. This component is unavailable in the recent versions of Bootstrap, but you can still use the official example for help.

Divide the home section into two parts. The left part contains a header, a paragraph, and a button similar to a link button. This button should navigate clients to the products section. The right part contains a cool image. You can use the image component in Bootstrap.

You should use the Lorem Ipsum dummy text for your site. It is a good idea to keep your website responsive. You can refer to the flexbox class to maintain the site's responsiveness.

In this stage, we are working on the home component ID.

You should add a wrapper <main/> tag inside the body after the <header/> tag. The rest of the components will be inside of the <main/> tag, except the <footer/> tag.

The layout of the HTML body for this stage should follow the structure below:

<body>
<header.../>
<main>
    <div (container)>
        <div id=home >
            <div (container)>
                <h1.../>
                <p.../>
                <a>
                    <button.../>
                </a>
            </div>
            <div (container)>
                <img.../>
            </div>
        </div>
    </div>
</main>
<script (Bootstrap)/>
</body>
Objectives

In this stage, your website should contain:

  1. An <h1> tag;
  2. A <p> tag with a dummy text;
  3. A button that navigates you to the products section;
  4. A cafe-related image.
  5. A <main/> tag as a wrapper in the body.
