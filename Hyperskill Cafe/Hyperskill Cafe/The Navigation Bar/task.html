Description

Most websites on the Internet have a navigation bar or navbar, for short. As you can understand from the name, it helps to navigate between different pages or parts of a website.

The navbar for this website will have navigation links for different sections of the website. The names of the sections are: home, about, products, reviews, contact. The navigation links are no different from the regular links; you just need to tweak them a little.

You should use the names of the sections as the values of href attributes.

Of course, the navbar should be responsive as well, but in this project, you need to use a design library called Bootstrap.

Use the official manual to add the Bootstrap library to your project and learn how to use the navbar component.

If you feel curious, you can also use this Net Ninja tutorial on YouTube to learn new things about Bootstrap.

Inside the body tag you should have two tags: <header/> and <script/>.

The navbar component should be wrapped by the <header/> tag.

The layout of the HTML body for this stage should follow the structure below:

<body>
<header>
    <nav>
        <div (container)>
            <a.../>
            <button>
                <span.../>
            </button>
            <div>
                <ul>
                    <li>
                        <a.../>
                    </li>
                    <li>
                        <a.../>
                    </li>
                    <li>
                        <a.../>
                    </li>
                    <li>
                        <a.../>
                    </li>
                    <li>
                        <a.../>
                    </li>
                </ul>
            </div>
        </div>
    </nav>
</header>
<script (Bootstrap)/>
</body>

Keep in mind that the above structure can only be considered as pseudocode and it is missing all the attributes and inner texts.

Don't worry if you make errors, the tests will guide you for the things you need to add.

Objectives

In this stage, your website should contain:

  1. The Bootstrap library linked with <link/> and <script/> tags;
  2. The Navbar component from Bootstrap;
  3. Several <a> links to navigate to different sections of the website.
  4. <header/> tag wrapping the navbar inside the body.
