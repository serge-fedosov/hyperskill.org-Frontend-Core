Description

The products section will contain a gallery of product images. The gallery is a Bootstrap grid with rows and columns. Use six product images related to various coffee shop items. Put the images inside a card component from Bootstrap to include a title and description underneath an image. Don't forget that the grid should adapt to different screen sizes.

In this stage, we are working on the products component ID.

Objectives

In this stage, your website should contain:

  1. A <h2> tag with the Our Products title;
  2. A grid layout;
  3. Six images wrapped in card components;
  4. Card description inside <p> tags;
  5. Card titles inside <h5> tags.
