Description

In the final stage of the project, we will work on the footer. You can take a look at this example from Bootstrap for help.

The footer is similar to the navbar component, but it is located at the bottom of your website. Like the navbar, it contains navigation links to different sections of the website. You can include various information in a footer but for this project, stick to the links and a copyright text.

The links should be inside an unordered list and wrapped as a list item.

The last link which is for the contact section should have a aria-current="page" attribute.

In this stage, we are working on the footer ID.

Objectives

In this stage, your website should contain:

  1. Various <a> links to different sections of the website;
  2. A <footer> tag;
  3. A copyright text;
  4. <ul> and <li> tags.
