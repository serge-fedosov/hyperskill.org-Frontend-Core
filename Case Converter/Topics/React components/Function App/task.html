Now that you know how to create and import a React component, it's time to put it all into practice.

Using your IDE inside src/index.js import the function App and use it inside ReactDOM.render .

Inside the file src/App.js create your function App , don't worry about the export, it's already defined for you. Inside the function App create a <div> element give it a class named main and inside the div a <h1> element and put some text inside.

To assign a class to a React element, you use the following syntax: className=""

Don't forget to run the command npm install in your terminal to install the dependencies before running the tests.
