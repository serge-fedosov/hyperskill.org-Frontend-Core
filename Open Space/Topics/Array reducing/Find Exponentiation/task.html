Find Exponentiation

Given an array of three elements, compute the exponentiation of the numbers from right to left. So, if the array is [x, y, z] you have to calculate (x ^ (y ^ z)). Use console.log in the function to output the reduced array.


Try reversing the order of the elements

Sample Input:
1 2 3

Sample Output:
1


Sample Input:
2 1 2

Sample Output:
2


Sample Input:
2 2 1

Sample Output:
4
